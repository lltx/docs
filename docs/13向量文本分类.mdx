> 智慧HR: 基于LangChain4j文本向量的面试者性格特征匹配

## 引言

在现代人力资源管理中，快速准确地评估候选人的性格特征对于找到最佳人才至关重要。本文将展示如何利用LangChain4j框架结合大语言模型，构建一个智能的面试者性格特征匹配系统。我们将从概念入手，详细介绍如何在Java中实现这一功能。

## 什么是性格特征匹配

性格特征匹配是HR领域的一项重要任务，它通过分析面试者的语言表达、行为模式等信息，将其与预定义的性格标签进行匹配。这有助于HR专业人士更好地评估候选人是否适合特定职位或团队文化。

## LangChain4j中的文本分类支持

LangChain4j框架通过`TextClassifier`接口，结合嵌入模型，为我们提供了实现性格特征匹配的强大工具。

### TextClassifier接口

```java
public interface TextClassifier<E extends Enum<E>> {

    /**
     * 对文本进行分类
     *
     * @param text 需要分类的文本
     * @return 文本所属的分类列表
     */
    List<E> classify(String text);

    /**
     * 对文本片段进行分类
     *
     * @param textSegment 需要分类的文本片段
     * @return 文本所属的分类列表
     */
    default List<E> classify(TextSegment textSegment) {
        return classify(textSegment.text());
    }

    /**
     * 对文档进行分类
     *
     * @param document 要分类的文档
     * @return 文本所属的分类列表
     */
    default List<E> classify(Document document) {
        return classify(document.text());
    }
}
```

### 基于嵌入模型的分类器实现

`EmbeddingModelTextClassifier`通过嵌入模型对文本进行向量化，并利用余弦相似度计算文本与性格标签之间的匹配度。

```java
public class EmbeddingModelTextClassifier<E extends Enum<E>> implements TextClassifier<E> {

    private final EmbeddingModel embeddingModel;
    private final Map<E, List<Embedding>> exampleEmbeddingsByLabel;
    private final int maxResults;
    private final double minScore;
    private final double meanToMaxScoreRatio;

    public EmbeddingModelTextClassifier(EmbeddingModel embeddingModel,
                                        Map<E, List<String>> examplesByLabel) {
        this(embeddingModel, examplesByLabel, 3, 0.7, 0.5);
    }

    public EmbeddingModelTextClassifier(EmbeddingModel embeddingModel,
                                        Map<E, List<String>> examplesByLabel,
                                        int maxResults,
                                        double minScore,
                                        double meanToMaxScoreRatio) {
        this.embeddingModel = embeddingModel;
        this.exampleEmbeddingsByLabel = embedExamples(examplesByLabel);
        this.maxResults = maxResults;
        this.minScore = minScore;
        this.meanToMaxScoreRatio = meanToMaxScoreRatio;
    }

    private Map<E, List<Embedding>> embedExamples(Map<E, List<String>> examplesByLabel) {
        return examplesByLabel.entrySet().stream()
                .collect(Collectors.toMap(
                        Map.Entry::getKey,
                        entry -> entry.getValue().stream()
                                .map(example -> embeddingModel.embed(example).content())
                                .collect(Collectors.toList())
                ));
    }

    @Override
    public List<E> classify(String text) {
        Response<Embedding> response = embeddingModel.embed(text);
        Embedding textEmbedding = response.content();
        List<LabelWithScore<E>> labelsWithScores = new ArrayList<>();

        exampleEmbeddingsByLabel.forEach((label, exampleEmbeddings) -> {
            double meanScore = 0;
            double maxScore = 0;
            for (Embedding exampleEmbedding : exampleEmbeddings) {
                double cosineSimilarity = CosineSimilarity.between(textEmbedding, exampleEmbedding);
                double score = RelevanceScore.fromCosineSimilarity(cosineSimilarity);
                meanScore += score;
                maxScore = Math.max(score, maxScore);
            }
            meanScore /= exampleEmbeddings.size();
            labelsWithScores.add(new LabelWithScore<>(label, aggregatedScore(meanScore, maxScore)));
        });

        return labelsWithScores.stream()
                .filter(it -> it.score >= minScore)
                .sorted((a, b) -> Double.compare(b.score, a.score))
                .limit(maxResults)
                .map(it -> it.label)
                .collect(Collectors.toList());
    }

    private double aggregatedScore(double meanScore, double maxScore) {
        return (meanToMaxScoreRatio * meanScore) + ((1 - meanToMaxScoreRatio) * maxScore);
    }

    private static class LabelWithScore<E> {
        E label;
        double score;

        LabelWithScore(E label, double score) {
            this.label = label;
            this.score = score;
        }
    }
}
```

## 实现性格特征匹配

### 依赖配置

在项目的`pom.xml`中加入必要的依赖：

```xml
<dependency>
    <groupId>dev.langchain4j</groupId>
    <artifactId>langchain4j</artifactId>
</dependency>

<dependency>
    <groupId>dev.langchain4j</groupId>
    <artifactId>langchain4j-open-ai</artifactId>
</dependency>
```

### 配置文件 `application.yml`

配置OpenAI的嵌入模型：

```yaml
langchain4j:
  openai:
    embedding-model:
      api-key: "your-openai-api-key"
      model-name: "text-embedding-ada-002"
```

### Ai Service实现

```java
@Service
@RequiredArgsConstructor
public class PersonalityMatchingService {

    private final OpenAiEmbeddingModel openAiEmbeddingModel;

    /**
     * 分析面试者的性格特征
     *
     * @param interviewText 面试文本
     * @return 匹配的性格特征列表
     */
    public List<PersonalityTrait> analyzePersonality(String interviewText) {
        EmbeddingModelTextClassifier<PersonalityTrait> classifier =
                new EmbeddingModelTextClassifier<>(openAiEmbeddingModel, PersonalityTraitExamples.examples);
        return classifier.classify(interviewText);
    }
}
```

### Controller实现

```java
@RestController
@RequestMapping("/api/hr")
@RequiredArgsConstructor
public class PersonalityMatchingController {

    private final PersonalityMatchingService personalityMatchingService;

    /**
     * 分析面试文本并返回匹配的性格特征
     *
     * @param interviewText 面试文本
     * @return 匹配的性格特征列表
     */
    @PostMapping("/analyze-personality")
    public List<PersonalityTrait> analyzePersonality(@RequestBody String interviewText) {
        return personalityMatchingService.analyzePersonality(interviewText);
    }
}
```
